J2Web-Immobiliare ver. 0.4.1.1


Requisiti: 
	Java Runtime Environment versione 7 o successive (non funziona con la versione 6 e precedente)
	E' necessaria una connessione ad Internet


Installazione: 
	semplicemente copiare la cartella j2Web all'interno del proprio computer


Avvio: 
	doppio click sull'icona del file eseguibile "j2web-0411"

Utilizzo:
	Sezione di sinistra - 
		compilare il  modulo per descrivere l'immobile da sincronizzare 
		(il pannello è suddiviso in 3 sottopannelli: info obbligatorie; immagini; info secondarie) 
		e creare la scheda cliccando su "Crea scheda"
	Sezione centrale - 
		dopo aver creato la scheda, questa può essere selezionata tra le disponibili nel pannello centrale, 
		selezionarla tramite la radio button apposita
	Sezione di destra - 
		dopo aver selezionato la scheda, si attiverà il pulsante "Inserisci" di ogni portale attivo. 
		Per inserire una scheda premere il pulsante "Inserisci". 
		Analogalmente per visualizzare una scheda inserita (o per cancellarla) premere "Visualizza" (o "Cancella")

	NB: una volta premuto "Inserisci" si avvia una serie di richieste al server del portale remoto che necessita di qualche secondo di attesa, 
	il cursore del mouse avrà lo stato di "in lavorazione", bisogna attendere qualche istante...
